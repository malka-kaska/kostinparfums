# Част 3 — Пълен план за Meta Ads кампания

## 3.1 Избор на формат
**Основен:** Advantage+ Catalog Campaigns
**Вторичен:** Carousel за редакторски колекции: "Top Sellers", "Signature by Occasion", "Dubai Collection".

## 3.2 Вълна + обективни карти
| Фаза | Обектив / Bid |
|-------|--------------|
| Хладно проспектиране | Sales / Advantage+ catalog automated |
| Посетители 24h | Sales / 20% cost cap → min ROAS |
| Добавено в количка / Checkout | Sales / min ROAS 4.0x |
| Минали купувачи | Sales / min ROAS 5.0x |
| Дубай/Oud хладно | Sales / Advantage+ catalog automated |

## 3.3 Бюджетно разделение (пример €3,500/месец)
- Хладно Adv+: €1,400 (40%)
- Посетители на каталог: €875 (25%)
- Карт спасителен: €700 (20%)
- Минали купувачи: €350 (10%)
- Редакторски Carousel A/B: €175 (5%)

## 3.4 Позициониране + таргетинг
- Позиционирания: Advantage+ auto (IG/FB/Reels/Messenger)
- Страни: BG, RO, HR, RS, GR, CY + EU/EEA
- Езици: български + английски; DE/IT/FR след локални PDPs
- Атрибуция: 7d click + 1d view

## 3.5 Три основни аудитории
1. Entry luxury: perfume/cologne + luxury goods + beauty
2. Aspirational men: men's fashion + premium accessories + grooming + designer followers
3. Dubai/Niche explorer: oud/arabic perfumes + Dubai/UAE travel + niche perfumery

## 3.6 Креативни куки
- TOFU: "2026-те са най-желаните лукс аромати"
- MOFU: "Ароматът, който ще носиш всеки ден"
- BOFU: "Количката изтича — запази бутилката си сега"
- UGC: "4.7★ от 200+ потвърдени купувачи"
- Editorial: "Завърши образа" кръстосани продажби

## 3.8 Pixel / CAPI events
`ViewContent`, `Search`, `AddToWishlist` (custom), `AddToCart`, `InitiateCheckout`, `Purchase`, `Lead`, `CompleteRegistration`, `ViewCategory` (custom), `DiscountApplied` (custom)

## 3.9 Изисквания към продуктов фийд
- Валута EUR, линк към exact PDP
- Полета: id/title/description/availability/condition=New/price/link/image_link/brand/gtin+mpn при наличие
- Custom labels: категория, best-seller ранг, безплатна доставка eligibility
- Синхронизация: real-time или два пъти дневно
- Снимки: чист неутрален фон, min 100x100px
